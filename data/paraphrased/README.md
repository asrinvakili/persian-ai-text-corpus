# Paraphrased Test Set — متن‌های آزمون بازنویسی‌شده

فولدر مربوط به آزمایش مقاومت در برابر بازنویسی (آزمایش ۵ در مقاله).

## محتوا

این فولدر هنوز خالی است. فایل `test_paraphrased.jsonl` با اجرای نوت‌بوک ۵ از مخزن [`persian-ai-text-detection`](https://github.com/asrinvakili/persian-ai-text-detection) تولید می‌شود:

```
experiments/05_robustness_paraphrasing.ipynb
```

پس از تولید، فایل به اینجا کپی کنید.

## ساختار رکورد

هر خط شامل همه فیلدهای متن مصنوعی اصلی به‌علاوه دو فیلد جدید است:

```json
{
  "id": "ai_0234_gpt",
  "human_id": "0234",
  "label": 1,
  "source": "ai",
  "model": "gpt-4o-mini",
  "category": "علوم نظری",
  "title": "نابرابری مارکوف",
  "text": "نسخه بازنویسی‌شده پس از ترجمه برگشتی",
  "original_text": "متن اصلی فارسی پیش از بازنویسی",
  "paraphrased": true,
  ...
}
```

نگه‌داشتن `original_text` به این دلیل است که خواننده بتواند تأثیر بازنویسی روی متن واقعی را بررسی کند، نه فقط روی متریک نهایی.

## روش تولید

- **مدل ترجمه فارسی → انگلیسی:** `persiannlp/mt5-small-parsinlu-opus-translation_fa_en`
- **مدل ترجمه انگلیسی → فارسی:** `persiannlp/mt5-small-parsinlu-translation_en_fa`
- **پارامترهای generate:** `num_beams=2`, `early_stopping=True`, `no_repeat_ngram_size=3`, `max_length=512`

## فقط متن‌های AI بازنویسی می‌شوند

متن‌های انسانی (۲۰۰ مورد در داده آزمون) دست‌نخورده باقی می‌مانند. این طراحی منعکس‌کننده سناریوی واقعی است: کاربر بدنیت می‌خواهد متن AI خود را پنهان کند، نه متن انسانی.

## ساعت‌شمار تخمینی

تولید کل ۶۰۰ متن بازنویسی‌شده روی T4 حدود ۳۵-۴۰ دقیقه زمان می‌برد. نوت‌بوک ۵ قابلیت ادامه از وقفه را دارد، پس می‌توان آن را به چند سشن تقسیم کرد.
